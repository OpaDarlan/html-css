Princípais formatações

Negrito / destaque

Nesta frase, tenho um termo em destaque usando a tag STRONG.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

e caso eu queira usar uma cor no fundo de outro texto, basta deixar apenas como MARK e criar um style lá em cima no head.

Texto deletado

podemos marcar um texto como deletado, para indicar que ele deve ser lido, mas não considerado.

texto inserido

podemos marcar um texto como inserido, para dar uma ênfase (deixar SUBLINHADO), e dizer qeue ele foi adicionado depois.

Texto sobrescrito

para inserir coisas do tipo x20+3 (digamos que ao quadrado)

Texto subscrito

para inserir coisas do tipo H2O (o 2 fica mais em baixo, ao contrario de ao quadrado)
